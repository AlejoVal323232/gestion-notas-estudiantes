# Sistema de Gestión y Calificación de Estudiantes

Este es un programa interactivo que desarrollé en C++ para registrar estudiantes, capturar sus datos, ingresar tres notas y calcular automáticamente sus promedios. Le agregué un menú interactivo para ver los resultados y comprobar el estado de aprobación de forma rápida.

## ¿Qué hace este programa?
* **Registro de datos:** Permite definir cuántos estudiantes se van a ingresar y guarda el nombre y el ID de cada uno.
* **Cálculo automático:** Procesa tres notas por estudiante y calcula el promedio final.
* **Menú interactivo:** Permite visualizar todos los resultados a la vez o ver el estado de aprobación ("Aprobado" o "Reprobado") usando un menú muy fácil de usar.

## Conceptos que demuestro aquí
En este código aplico las bases de la programación estructurada en C++:
* **Estructuras de datos (Arreglos paralelos):** Uso arreglos para guardar y relacionar los nombres, IDs y promedios.
* **Estructuras de control:** Uso ciclos for para recorrer y procesar la información de los estudiantes en orden.
* **Menús interactivos:** Implemento un ciclo do-while con un switch-case para que el usuario navegue por las opciones y para validar lo que ingresa.
* **Lógica condicional:** Utilizo bloques if-else para evaluar los promedios y definir si el estudiante aprueba o no.

## Cómo ejecutar este programa

* Asegúrate de tener instalado un compilador de C++ como **Dev-C++**.
* Abre el archivo `gestion_notas.cpp` en tu editor.
* Compila y ejecuta el código.
